Цель разработки HTML5 — улучшение уровня поддержки мультимедиа-технологий с одновременным сохранением обратной совместимости, удобочитаемости кода для человека и простоты анализа для парсеров.</p> <h2>Новые API</h2> <p>В дополнение к определению разметки HTML5 устанавливает API[5][33], который может быть использован с JavaScript.
